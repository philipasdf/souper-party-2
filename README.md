# Souper Party

Open Souper Party in your browser: https://souper-party.web.app/

## local installation
* install npm
* install angular cli
* install firebase tools

## Development server
```
npm start
```
Navigate to `http://localhost:4200/`.


## Build
```
ng build --prod
firebase login
firebase deploy
```
The build artifacts will be stored in the `dist/` directory.
